## Freelance Scrapper

A simple service that scrapes freelance orders from multiple websites and returns data.
Perfect for aggregators, bots, dashboards, and any apps that need freelance orders.

---

## Features

- Scrapes **freelance websites**
- Returns clean output
- Clear modular project structure

---

## Installation (local)

**1. Clone the repository**
```bash
git clone https://github.com/EmilioAugust/FreelanceScrapper.git
cd FreelanceScrapper
```
**2. Install dependencies**
```bsah
pip install -r requirements.txt
```
**3. Run the program**
```bash
cd app
python consumer.py
```
In new terminal:
```bash
cd app
python main.py
```
